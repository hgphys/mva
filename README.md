# 多変量解析

金沢学院大学 情報工学部情報工学科「多変量解析」（担当: 後藤 弘光）の講義ノート・演習教材リポジトリです。

---

## 授業概要

複数のデータ変数の間の相互関係を分析する統計的技法を多変量解析と呼ぶ。多変量の複雑な関係を簡単に理解するために、分析者は適切なモデルを用いて分析を実行する必要がある。本講義では多変量解析の主要な手法について、理論的な理解とともに、実際のデータを用いての具体的な分析を実践し、多変量解析の活用法を身に付ける。

## 到達目標

- 多変量解析の主要な統計モデルについて理解する
- Python を用いて主要なモデルによる分析を実行する
- 多変量データから統計的に重要な特徴量を抽出する技術を身に付ける

> **本講義の特徴**: 上記の各手法を個別の道具としてではなく、「目的関数 $J$ を定義し、それを最大化または最小化するパラメータを求める」という **数理最適化の問題として統一的に**扱います。データを「$n$ 行 $d$ 列の行列」として捉え直すことで、多変量解析が **行列演算（線形代数）** そのものであることを体験し、アルゴリズムの脱ブラックボックス化を目指します。全 15 回は最適化の目的に応じて次の 6 つの Part に整理されます。
>
> | Part | 対象回 | テーマ | 最適化の視点 |
> | :--- | :--- | :--- | :--- |
> | I | 第1〜3回 | 座標と情報 | 公平な比較座標の作成 |
> | II | 第4〜5回 | 誤差の最小化 | 予測誤差の最小化（重回帰分析など） |
> | III | 第6〜7回 | 確率の最大化 | 尤度の最大化（ロジスティック回帰など） |
> | IV | 第8〜9回 | 凝集度の最適化 | 群内距離の最適化（クラスタ分析） |
> | V | 第10〜11回 | 分離の最大化 | 群間分離の最大化（判別分析など） |
> | VI | 第12〜14回 | 情報の最大化 | 分散の最大化による圧縮（主成分分析など） |
>
> 各回のノートブックには、目的に応じて 2 種類の演習課題を用意しています。
> - **A（数理理解）**: Python を「高度な電卓」として使い、線形代数・微分・統計の計算プロセスを追体験します
> - **B（応用・実装）**: `pandas` や `scikit-learn` などの主要ライブラリを活用し、実際のデータ分析フロー（可視化・モデリング・解釈）を体験します

## 実行環境

本講義は **Google Colaboratory** での受講を前提としており、演習データは Google スプレッドシートから直接読み込む形式になっています。ブラウザだけで完結するため、ローカルでの環境構築は不要です。

1. このリポジトリをクローン（または ZIP ダウンロード）する
   ```bash
   git clone https://github.com/hgphys/mva.git
   ```
2. [Google Colab](https://colab.research.google.com/) にアクセスし、「ファイル」→「ノートブックをアップロード」で `.ipynb` を開く
3. ノートブック内の指示に従ってセルを実行する（データの URL は各ノートブック内に記載）

### ローカルで実行したい場合

ローカル環境（Python 3.11 以上）でも実行可能です。`requirements.txt` を参考に必要なパッケージをインストールしてください。

```bash
conda create -n mva_env python=3.11 -y
conda activate mva_env
pip install -r requirements.txt
jupyter lab
```
